# 週間時間割

非公開のGoogleスプレッドシートを、各利用者本人のGoogleアカウント権限で読み取る閲覧専用Webアプリです。

## 起動

```sh
npm start
```

ブラウザで `http://127.0.0.1:4173` を開きます。

## Google連携の設定

1. Google Cloud Consoleでプロジェクトを作成します。
2. Google Sheets APIを有効にします。
3. OAuth同意画面を設定します。
4. 「ウェブ アプリケーション」のOAuthクライアントIDを作成します。
5. 承認済みのJavaScript生成元に、本番URLと開発用の `http://127.0.0.1:4173` を登録します。
6. `config.js` にクライアントID、スプレッドシートID、取得範囲を設定します。

クライアントシークレットは使用しません。利用者はそれぞれGoogleにログインし、自分が閲覧できるシートだけを取得できます。

## シート形式

現在は月間シートの1週目を表示します。B〜F列を月〜金として読み取ります。

| 時限 | 授業名の行 | 内容の行 |
| --- | ---: | ---: |
| 1限 | 6 | 7 |
| 2限 | 10 | 11 |
| 3限 | 15 | 16 |
| 4限 | 19 | 20 |

授業名が空欄の場合、その直下に内容が入っていてもアプリには何も表示しません。
