# 週間時間割

非公開のGoogleスプレッドシートを、各利用者本人のGoogleアカウント権限で読み取る閲覧専用Webアプリです。

## 起動

```sh
npm start
```

ブラウザで `http://127.0.0.1:4173` を開きます。

## Google連携の設定

1. Google Cloud Consoleでプロジェクトを作成します。
2. Google Sheets APIを有効にします。
3. OAuth同意画面を設定します。
4. 「ウェブ アプリケーション」のOAuthクライアントIDを作成します。
5. 承認済みのJavaScript生成元に、本番URLと開発用の `http://127.0.0.1:4173` を登録します。
6. `config.js` にクライアントID、スプレッドシートID、取得セル範囲を設定します。

クライアントシークレットは使用しません。利用者はそれぞれGoogleにログインし、自分が閲覧できるシートだけを取得できます。

初回にGoogleの閲覧許可を行った後は、Googleにログイン済みであれば次回以降の認証を自動復元します。Googleからログアウトした場合や許可を取り消した場合のみ、再ログインが必要です。

## シート形式

`1月`、`2月`、`3月` のような月名のシートを自動検出し、存在する月だけを取得します。後から月別シートが追加された場合も、アプリの「更新」で自動的に反映されます。8月は対象外なので、7月の次は9月になります。

各月の5週分を読み取り、現在の日付に対応する週を最初に表示します。画面上の「前の週」「次の週」で月をまたいで切り替えられます。B〜F列を月〜金として読み取ります。画面タイトルは選択中の週に合わせて `6月15日〜19日` の形式で表示します。

| 時限 | 授業名の行 | 内容の行 |
| --- | ---: | ---: |
| 1限 | 6 | 7 |
| 2限 | 10 | 11 |
| 3限 | 15 | 16 |
| 4限 | 19 | 20 |

授業名が空欄の場合、その直下に内容が入っていてもアプリには何も表示しません。

2週目以降は21行間隔です。日付行は3・24・45・66・87行で、授業名と内容の行にも週ごとに21を加算します。

例外として、6月17日の3限・4限は授業なしとして表示します。

スマートフォンでは授業名を略称で表示し、授業セルをタップすると正式な授業名と内容を画面下の詳細パネルに表示します。
